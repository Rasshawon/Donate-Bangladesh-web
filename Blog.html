Important FAQ

what is the Document Object Model (DOM)?

A programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?

The getElementById() method of the Document interface returns an Element object representing the element whose id property matches the specified string.

what is event delegation in the context of the DOM and why is it useful?

Event delegation refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you manipulate an elements attributes and styles using the DOM?

we can use element.classList.add(), element.setAttribute() ,element.style.attribute etc to manipulate the elements attributes and styles.
